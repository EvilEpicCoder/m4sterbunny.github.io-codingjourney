How to create new page

Basics

3 types of Pages:

  • Post
    • Post is like news
      hexo new Hello world!
  • Draft
    • Draft unpublished news
      hexo new draft Second hidden hello world!
    • Make it published:
      hexo publish Second hidden hello world!
  • Pages
    • Menu like pages
      hexo new page About

Examples:

This is path in console where it create new page

hexo generate
