Setup to upload to github pages

  • Install node.js here https://nodejs.org/en/
  • Install git here https://git-scm.com/
  • Installation hexo-cli on PC $ npm install hexo-cli -g

How to create new page

Basics

3 types of Pages:

  • Post
    • Post is like news
      hexo new Hello world!
  • Draft
    • Draft unpublished news
      hexo new draft Second hidden hello world!
    • Make it published:
      hexo publish Second hidden hello world!
  • Pages
    • Menu like pages
      hexo new page About
